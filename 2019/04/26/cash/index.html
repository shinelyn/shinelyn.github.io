前端缓存

缓存的分类
按位置：（memory cache，disk cache，service worker）
按失效策略：（Cache-Control，E-tag）

存贮位置的优先级 1.service worker 2.Memory Cache 3.Disk Cache 4.网络请求 找到即返回。

Memory Cache (内存中的缓存)
所有的网络请求资源最后都会被浏览器加载到 Memory Cache中，常规情况下，浏览器的 TAB 关闭后该次浏览的 memory cache 便告失效
